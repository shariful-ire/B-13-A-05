1️⃣ Difference between var, let, and const

var, let, and const are used to declare variables in JavaScript.
var is the old way and it can be redeclared and updated.
let is a modern way and it can be updated but not redeclared in the same scope.
const is used when the value should not change, so it cannot be updated or redeclared.

2️⃣ What is the spread operator (...)

The spread operator ... is used to expand elements of an array or object.
It helps to copy, merge, or pass multiple values easily.

Example:
const newArray = [...oldArray];

3️⃣ Difference between map(), filter(), and forEach()

map() is used to create a new array by transforming each element.
filter() is used to create a new array with elements that match a condition.
forEach() is used to loop through an array but it does not return a new array.

4️⃣ What is an arrow function

An arrow function is a shorter way to write a function in JavaScript using =>.

Example:
const add = (a, b) => a + b;

5️⃣ What are template literals

Template literals are used to write strings with variables easily using backticks ` `.
It allows embedding variables inside strings using ${}.

Example:
`Hello ${name}`
